# Anshika Jaiswal — Portfolio Website

A modern, responsive developer portfolio built with React, TypeScript, and Three.js.

🌐 Live: https://anshika-portfolio-tcov-2m42f0tmv.vercel.app/

## ✨ Features

- Interactive 3D animations with Three.js
- Smooth transitions with GSAP
- React + TypeScript codebase
- Career timeline & experience section
- Projects showcase
- Certifications & achievements
- Fully responsive (desktop + mobile)

## 🧰 Tech Stack

- React, TypeScript, Vite
- Three.js / WebGL
- GSAP
- Tailwind CSS / CSS Modules
- Deployed on Vercel

## 🚀 Run Locally

1. Clone the repo
```bash
   git clone https://github.com/Anshika1772/anshika-portfolio.git
   cd anshika-portfolio/portfolio-website
```

2. Install dependencies
```bash
   npm install
```

3. Start dev server
```bash
   npm run dev
```
